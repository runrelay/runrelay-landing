API

One call. Full execution.

Send a client intent via REST API. RunRelay handles search, booking, payment, ticketing, and post-booking changes ‚Äî all through a single integration.

// Create a travel request
const response = await fetch('https://api.runrelay.io/v1/request', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    client_id: 'usr_29xKj3',
    intent: 'flight',
    details: {
      from: 'DXB',
      to: 'LHR',
      date: '2026-03-15',
      class: 'business',
      passengers: 1
    }
  })
});

// Returns: options, pricing, booking link
const data = await response.json();
